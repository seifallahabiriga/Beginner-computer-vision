# Setup Instructions

1. Install dependencies

2. Spotify Developer App:
  Go to Spotify Developer Dashboard
  Create a new app.
  Copy Client ID and Client Secret.
  Add a redirect URI: http://localhost:8888/callback

3.Environment Variables:
  Create a .env file in the project root:
    SPOTIFY_CLIENT_ID=your_client_id_here
    SPOTIFY_CLIENT_SECRET=your_client_secret_here
    SPOTIFY_REDIRECT_URI=http://localhost:8888/callback

4. Get Authorization Code:
Run the local server provided by Spotipy or your auth script:
bash
python server.py
Follow the link in terminal, login to Spotify, and copy the returned code.

5. Generate Token:
Use get_code_for_autorization.py (or the provided script) to exchange the auth code for an access token.
Store token securely or in .env.

6.Refresh Token:
Tokens expire; call the refresh function to get a new token automatically before API calls.
